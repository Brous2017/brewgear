BrewGear is a simple application for calculating beer recipes.

Features:

* Works straight from your browser
* No internet connection required: HTML 5 storage API is used for client side storage.
* Malt, hop, water calculation
* Primary language is Dutch (anybody wants to translate?)
* SI units (grammes, litres)
* Works on Safari (out of the box), Firefox (with gears plugin) and Chrome

No Internet Explorer? Well, no.

This package contains a HTML page + scripts to calculcate beer recipes. The
page is in Dutch, since that's the native language of the author.

The remainder of this document is written in Dutch too.
